# ARTI-308_lab10
This lab focuses on building and evaluating a Support Vector Machine (SVM) model using the famous Iris flower dataset. The dataset contains 150 flower samples from three Iris species: Setosa, Versicolor, and Virginica, with measurements of petal and sepal length and width.

In this lab, I:

Explored and visualized the Iris dataset

Split the data into training and testing sets

Trained an SVM classification model using Scikit-Learn

Evaluated the model using confusion matrices and classification reports

Applied GridSearchCV to tune hyperparameters and improve model performance

The lab demonstrates the fundamentals of supervised machine learning, classification, model evaluation, and hyperparameter optimization using Python and Scikit-Learn.
